# Data Analyst Practice Portfolio

This repository contains beginner data analysis practice projects focused on cleaning, organizing, querying, and reporting data.

## Objective

Build practical data analysis skills that connect to business operations, help desk reporting, and security data analysis.

## Tools Used

- Excel
- SQL / MySQL
- CSV datasets
- Dashboards
- Pivot tables
- Basic reporting workflows

## Project Areas

### Excel Analysis

- Clean messy data
- Build pivot tables
- Track KPIs
- Create simple dashboards
- Summarize operational trends

### SQL Analysis

- Select and filter data
- Sort and group results
- Join tables
- Count events or tickets
- Build reports from structured data

### Business Operations

- Delivery performance tracking
- Attendance tracking
- Package count reporting
- Productivity summaries

### Security Data Analysis

- Failed login trends
- User activity review
- Event counts by host or source IP
- Basic SOC reporting
